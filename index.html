RLO: Reusable Learning Objects

This website was created as a school project on RLO or reusable learning objects. Its purpose is to teach basic jQuery features such as

  • Installing jQuery on your next web project
  • Selecting elements with jQuery selectors
  • Binding behaviors to elements
  • Adding animation
jQuery is a powerful, lightweight JavaScript library that enables rapid web development. “jQuery achieves this goal by making common, repetitive, tasks, stripping out all the unnecessary markup, and leaving them short, smart and understandable”. In addition, it provides beautiful effects and animations that will make your website more interactive, intuitive, and alive. Lastly, jQuery is CSS3 compliant and is cross-browser compatible.

That has been said, lets jump into the tutorial and discover how you can integrate jQuery into your next web project.
