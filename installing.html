Installing jQuery

  • Introduction

  • Conclusion

Introduction

jQuery library comes in two formats, compressed and non-compressed. The compressed version is smaller in size thus faster to download. This version is recommended for production ready web application. On the other hand the uncompressed version is larger and it is good for debugging your web application.

jQuery library can be integrated into your web application via the <script> html tag. You can either use a local copy of the jQuery library, downloadable from www.jquery.com or reference to one of the available CDNs.

Use Local Copy

- To use a local copy of the jQuery library you need to visit www.jquery.com and either download the minified (compressed version) or the uncompressed version.

- Save the jQuery library into a folder/directory within your web application. Make sure that this folder is accessible by everyone.

For example, below is a simple folder structure of a website application

  • Root HTdocs
  • Applications
  • css
  • images
  • javascript
    • jquery-1.7.1.min.js
    • jquery-1.7.1.js
  • index.html

- From your web application add a <script> html tag and refers its src attribute to the jQuery library. 

  <!DOCTYPE html>
  <html lang="en">
    ...
    <script type="text/javascript" src="/javascript/jquery-1.7.1.min.js"></script>
    ...
  </html>

Use CDN

- CDN is also known as content delivery network. "It is a large distributed system of servers deployed in multiple data centers in the Internet". The purpose of using a CDN hosted solution is because its high availability and high performance.

- Some of the common hosted jQuery CDN solutions are

  • Google CDN: http://ajax.googleapis.com/ajax/libs/jquery/1.7.1/jquery.min.js
  • Microsoft CDN: http://ajax.aspnetcdn.com/ajax/jQuery/jquery-1.7.1.min.js
  • jQuery CDN: http://code.jquery.com/jquery-1.7.2.min.js

- More information on other hosted CDN solutions could be found at http://www.cdncatalog.com/

- From your web application add a <script> html tag and refers its src attribute to the hosted CDN jQuery library. 

  <!DOCTYPE html>
  <html lang="en">
    ...
    <script type="text/javascript" src="http://ajax.googleapis.com/ajax/libs/jquery/1.7.1/jquery.min.js"></script>
    ...
  </html>
