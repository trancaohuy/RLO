Introduction

When it comes to handling and triggering events jQuery provides great utilities for this. It allows developers to bind to elements' events as well create and manage new custom events. In this tutorial we will take a look at some basic features that jQuery offers.

Bind to an event

jQuery utilized a generic method to bind a handler function to an event. This allows developers to subscribe a handler to an element's events to create a dynamic web application.

To bind a handler to an event you will need to identify a number of elements

  • Identify which element you would like to query against
  • The name of the element's event you would like to bind a handler
  • Identify what your handler will do when it catches an event

For example: To bind to a click event of an <a> tag we would query the element and use the bind function to assign a handler to a particular event. 

  $('a.myATag').bind('click',
  function(){
    alert('myATag triggers a click event');
  });

The above example illustrates the following

  • We query all <a> elements with a class attribute equals to myATag
  • Next we bind to the element's click event
  • We then assigned a handler as the second parameter of the bind method. This handler will alert a message into the browser.

We then bind each element click event to a custom handler. This custom handler creates a message box with the text "myATag triggers a click event". Lastly, we run the return false; to override the element's normal behavior.

There are many other element events that you could bind a handler with. Some of the most used events are click, focus, blur, etc. More information could be find at http://api.jquery.com/category/events/

Creating custom event

One of the most powerful features of jQuery and tend to be the forgotten one is the ability to create custom event. With this feature developers can create and consume custom events or build custom plugins that require the process of publish and subscribe events.

To create a simple custom event you will need to tell jQuery that you want to create an event. With this event object you could bind to a HTML element and publish the event on its behalf. 

//Create a custom event
var myCustomEvent = jQuery.Event('customEvent');

// fires the custom event
jQuery("myElement").trigger(myCustomEvent);

Conclusion

jQuery has an easy to use, extensible, and scalable library that works with events. It allows you to bind your custom handler code to an event with a line of code. In addition, it makes creating and consuming custom events a joy to code. More information on jQuery events could be found at http://api.jquery.com/category/events/
