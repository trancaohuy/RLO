Introduction

At the core of jQuery is its abstraction layer for querying elements and objects within an HTML document. Using jQuery developers could select one or more DOM elements using either the jQuery('selector'[, context]) syntax or the shortcut syntax $('selector'[, context]) and manipulate the elements with a couple of lines of code.

Both methods required that the selector is either

a string selector
an object
an HTML element
an array
jQuery object

The context object refers to the context where jQuery should query for the elements from. The default value for this object is the HTML document. Other values that can be use for the context object are

a string selector
an object
an HTML element
jQuery object

In this tutorial we will discuss a number of ways developers could use to select DOM elements from an HTML document.

Selectors on an element ID

Each HTML elements could be assigned with a unique ID via the id attribute. It is this value that uniquely identifies the element within a HTML document.

For example: <div id="myID">Hello world</div>

To query an element with a particular id in jQuery we use the following syntax

$('#elementID')

The selector value here is a string that composed of a hash value # follows by the id of the element you are querying.

For example: $('#myID')

The result of the query will be an array of elements that matches the selectors.

Selectors on Element's Class Attribute

With CSS developers were able to template their website thus allowing them to reuse their code as well adopt a consistent look and feel. To do this we assign HTML elements with a class value. From there developers could declare a look and feel for a particular class in the css file.

For example: <div class="successMessage">Transaction completed successfully !!!</div>

In order for developers to retain this consistency once the page has been rendered jQuery provides a selector mechanism that queries all the elements using the class attribute.

To query all elements with a particular class in jQuery we use the following syntax

$('.classValue')

The selector value here is a string that composed of a period . follows by the class value.

For example: $('.successMessage')

The result of the query will be an array of elements that matches the selectors.

Selectors on CSS Element

jQuery also has the ability to select elements that match a particular CSS attribute. Although, it doesn't provide an abstraction method for this we could utilize jQuery filter method to achieve this behavior.

For example: <div style="font-size: 10px;">Transaction completed successfully !!!</div>

To query all elements with a CSS font-size of 10 pixels we would use the syntax below

$('div').filter(function(){ return $(this).css('font-size') == '10px' })

The syntax specified above instructed jQuery to retrieve all div elements within the actual-markup container and return a subset of the result base on the element's CSS font-size attribute.

The result of the query will be an array of elements that matches the filter.

Selectors on Element's Attribute

In addition, jQuery supports the ability to query HTML elements base on the attribute's value

For example: <div bindClass="myBindClass"</div>

The syntax for such query is $('[attribute="value"]') where attribute refers to the attribute's name and value refers to the attribute's value

To query all elements with an attribute name of bindClass and the attribute's value of myBindClass we would use the syntax below

$('[bindClass="myBindClass"]')

The result of the query will be an array of elements that matches the selectors.

Selectors on HTML tag

Lastly, jQuery provides the support to query elements base on its HTML tags.

For example: <p>Test paragraph</p>

The syntax for such query is $('tagName') where tagName refers to the element's tag name

To query all p elements we would use the syntax below

$('p')

The result of the query will be an array of elements that matches the selectors.

Conclusion

At the core of jQuery is its ability to simplify the process of querying DOM elements from a HTML document. The tutorial above captures a subset of selectors functionalities that come with jQuery. More information could be found at http://api.jquery.com/category/selectors/
