Conclusion

jQuery library is one of the most widely used javascript libraries. It includes many different utilities that helps developers speed up their development time, create interactive websites, easy to use and highly extensible.

In addition to the different topics addressed in this tutorial there are many other jQuery features and plugins that developers can integrate to their web application.

  • Animation and effects: jQuery comes with a nice toolbox of animation utilities. More information on these utilities can be found at http://api.jquery.com/category/effects/
  • Ajax utilities: Modern web applications utilized Ajax methodology to make web applications more interactive. They achieved these interactivity by creating asynchronous calls to the server to fetch and render data. This methodology prevents a full refresh of the web page thus allowing the users to have better interaction with the application. In addition, this methodology put less stress on the server because only a subset of the page need to be refresh. More information on Ajax support can be found at http://api.jquery.com/category/ajax/
  • Widgets: jQuery and its opensource community has created a nice set of widgets that developers can integrate into their web application. Widgets such as Calendar, Lightbox, Sliders, etc can be found at jQueryUI.
  • Theming: In addition to the different widgets that developers can use jQuery also provides a set of themes that would instantly change the look and feel of your web application. More information can be found at jQuery Theme.
